Colorize
========
Colorize is a simple and lightweight tool aimed to take one color format and convert it to another. 

### Prerequisites
* .NET 4.7.1

### Download
Colorize can be downloaded [here](https://github.com/InspireNXE/Colorize/releases/latest).
